# CineRecall Trivia Assets

This folder should contain **composite puzzle images** for the trivia game.

## Required Images

Each puzzle uses a **single composite image** that combines the rebus clues:

| File | Movie | Description |
|------|-------|-------------|
| `puzzle_vidhi.png` | Vidhi (Rajinikanth) | Bell + Law Hammer = Courtroom theme |
| `puzzle_padagotti.png` | Padagotti (MGR) | Boat + Face = "படகோட்டி" |
| `puzzle_thevar.png` | Thevar Magan (Kamal) | Eye + Nail + King composite |

## Image Guidelines

For elderly-friendly visibility:
- **Dimensions:** 1024x768 pixels (landscape) or 768x1024 (portrait)
- **High contrast** colors with clear visual elements
- **Simple illustrations** - avoid complex photographs
- **Solid backgrounds** (preferably cream/beige #F7F5F2)
- **Bold outlines** for easy recognition
- **Large icons** within the composite

## Composite Image Tips

1. Arrange rebus elements horizontally with "+" symbols between them
2. Add "= ?" at the end to indicate it's a puzzle
3. Use consistent styling across all puzzle images
4. Add subtle shadows to make elements pop

## Accessibility Notes

- Images should be self-explanatory even without color
- Use icons familiar to Tamil-speaking elderly audience
- Test visibility at 50% zoom to ensure readability
- Consider adding Tamil labels within the image
